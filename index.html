PuppetTrack

kinect v2 based object tracking system

Download this project as a .zip file Download this project as a tar.gz file

Welcome to huangchaodian Pages.

we present an speech rehabilitation training system based on kinect2.0.Instead of tracking skeleton,we proposed a system tracking the motion of the user’s familiar rigid toys,props,and puppets.The user can manipulates these objects directly in front of an inexpensive Kinect 2.0 depth camera and our system renders the corresponding 3D models into a virtual set in real-time.However the user may lack of fine motor skills,the object size matters to them more than normal children.It is thus vary important to chose a suitable size of a tangible objects such that it can be grabbed with ease[TUI..],which sometimes means vary small.The small size objects challenge the real-time tracking system in several aspects.First of all,raw data size of the object to be tracked captured by Kinect2.0 will also be small(sometime less then 200 points),which make it hard for the algorithm to use a local feature extract method(i.e,SIFT).Also the data is noisy,the algorithm must be robust to the noise data. Our method overcomes these challenge and run in real time.
